# Executive Reporting Dashboard

Separate Streamlit project for executive-level delivery and portfolio reporting.

## Run
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

## Purpose
- Program leads update recurring delivery, milestone, and health information.
- Executive stakeholders consume a polished reporting view without the workflow-editing surface.
- This project is intentionally separate from the analyst workflow dashboard so both apps can evolve independently.
